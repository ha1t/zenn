# XAMPP

10年以上前からPHPを知っている人なら名前くらいは聞いたことがある、PHPとその周辺プロダクトを一括でインストールできるパッケージソフトです。

https://www.apachefriends.org/jp/index.html

昔はMySQLが入っていたのですが、今は代わりにMariaDBが入ってるみたいです。初心者の使用感としては変わらないので今でも使えますね。

## Install

昔は公式サイトからmsiをダウンロードしてきて、とかありましたが、今はスタートボタンを右クリックしてターミナルを起動したら以下を入力するだけです。

```bash
 winget install ApacheFriends.Xampp.8.2
```

スタートメニューにアイコンが追加されるので、それをクリックすれば起動します。

これ、初学者の選択肢として悪くないんじゃないかなと思います。が、本書は「PHPを使ってプログラミング入門する」というスタンスなので、Laravelを使ったりDBを使ったりというPHPらしい入門とはまたちょっと違うんですよね。そういう意味ではXAMPPはちょっとヘビィなのかなと思いました。

# WSL

## Install

まずはWSL上で動くLinuxを入れないといけなくて、この遠回り感が初心者や初心者向けに解説本を書く人にはつらいかも。
Microsoft Storeアプリを起動してUbuntuを探していれるか、

```bash
 winget install Canonical.Ubuntu.2404
```

でもインストールできます。

そしたら Windows TerminalからWSLを起動して、あとはLinuxなので `sudo apt install php` などすればいいんですが、やはり導入までの手順が多くて手間を感じますね。

私は最近WSLの便利さに気づいて、目的別に複数のイメージを作りdevcontainerで開発するスタイルに落ち着いたのである程度知識があるならWSLをオススメしたいです。

# PHP For Windows

今回の目的にはこれが一番最速なんじゃないかと思いました。

## Install

PHPの公式ページにWindowsでアクセスしてDownloadページにいくと自動的にバイナリを落とす画面に誘導してくれます。

コマンドラインでもインストールできて、すごく簡単です。

```bash
 winget install PHP.PHP.8.4
```

PHPしか入らないのでDB接続したいなら別途MySQL入れないといけないし、ブラウザからみたいならApacheやnginxがないので、`php -S` でしのぐ必要があります。

が、本書の内容であればDB接続もなく、要求されるWebの表示もシンプルなのでこれだけでいけます。実際に本の中でもwebサーバを別途インストールするようなことはせずに `php -S` で動かしています。

# まとめ

昔のwindowsはPHPを動かすのに手間がかかっていたけど、winget環境が充実したことでわりと簡単にセットアップできるようになりました。
本書ではこのあとGoogle Chromeを入れたりVSCodeを入れたりしますが、これらもwingetでインストールできます。

Macが買えなくてPHPでプログラミング入門をあきらめていた人は是非Windowsでも簡単にできるのでチャレンジしてください。

# PHPカンファレンス小田原2026参加ゆるレポート

https://phpcon-odawara.jp/ja/2026/

所属企業がスポンサーをやっており、ボスから「スポンサーやってるからブースだすけど来る？」といってもらえたので、ブースに待機できる人間は多いほうがいいよねということで参加してきました。


# 小田原って遠いけど近い

![](/images/20260416_01.webp)

そもそもあまり外出しない側の人間なので、小田原にピンと来てなかったのですが、東京駅から電車一本で行けて、小旅行気分を味わえる絶妙な距離でした。
電車一本ということは特に迷うこともなくたどり着けるのもアドですね。
一泊して観光してもよかったなー、と思ったのですが、実際地方イベントに参加慣れしている方々は前日や後日を観光にあてているようです。

![](/images/20260416_02.webp)

会場は、駅から歩いてすぐの交流センターでした。近くにコンビニもあって、長時間でも快適に過ごせました。


# コラボランチ

PHPカンファレンス小田原がコラボしているお店が何店舗かあるということでびっくり。会場からも近く、行ってみると店舗入口にはPHPカンファレンス小田原の張り紙が。

![](/images/20260416_03.webp)

いくつかあるコラボ店舗の中で今回は昇玉さんでご飯を食べたのですが、麻婆豆腐が本当においしかったです。
遠方から来ると、周りにどんなお店があるかわからない状態だと思うので、こういう仕組みがあるのはありがたいです。

![](/images/20260416_04.webp)

知らない場所で知らないお店に挑戦するのはリスクが高くてコンビニですませるか。ってなる人も多いと思うんですよね。
運営からオススメのお店を教えてもらえるのは助かる。

# うめまる

![](/images/20260416_05.webp)

カンファレンス終盤、突然のご当地ゆるキャラの登場にほっこりw
BGMも「おだわら梅丸音頭」でクセになる感じでした。小田原は梅の産地だということを学びました。

PHPと小田原を学べるカンファレンス、おもしろいなあ。


# スポンサー目線で嬉しかったこと

## スポンサー紹介に時間をつくってくれた

個人スポンサーも含めてちゃんとスライド内で紹介されており、シンプルにスポンサーでよかったなと思いました。こういった細かい配慮が行き届いており、非常に丁寧に運用されているなと感じました。

## スタンプラリー

今回はスポンサー枠で参加しているので、ほぼほぼブースにいたのですが、スタンプラリーの仕組みがあるおかげで、たくさんの人達にブースに来てもらえました。
東京のPHPカンファレンスでもやってますが、この仕組みがあると参加者の人たちに名前を覚えて帰ってもらえるのでありがたいですね。

## ピッチとLT

これも覚えて帰ってもらえる仕組みですよね。弊社の社長がLTをしてひと笑いとってくれました。

私は法人そのものに興味があるタイプの人間なので、スポンサー企業がどんな業務をやっているのかなどを知ることができて楽しかったです。


# 地方でカンファレンスが開催される理由がわかった

地方のカンファレンスへの参加は初めてだったのですが、自分がよく参加していたカンファレンスとはまったく違うあたたかい雰囲気を感じられて「参加してよかったなー」と思えました。

すごいエンジニアのすごい話を聞きにいく、最先端の技術を学びにいく。というもの以外にもたくさんのものを得ることができるのがPHPカンファレンス小田原なんですね。

実行委員長の @asumikam さん、素晴らしいカンファレンスを開催していただきありがとうございました。

他の方のブログ記事は https://fortee.jp/phpconodawara-2026/blog-link から参照できます。
